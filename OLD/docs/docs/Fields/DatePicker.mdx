# DatePicker

Generate a datepicker field based on [react-datepicker](https://reactdatepicker.com).

## Options

```js
DatePicker('fieldName', {
  label: 'Label of the field',          // (optional)
  help: 'Some additional informations', // (optional)
  time: false,                          // default to false (optional)
  default: 'Default value',             // default to '' (optional)
})
```
