# Select

Generate a select field

## Options

```js
Select('fieldName', {
  label: 'Label of the field',          // (optional)
  help: 'Some additional informations', // (optional)
  default: 'option1',                   // default value to 0 (optional)
  options: [
    {
      value: 'option1',
      label: 'Option 1'
    }, {
      value: 'option2',
      label: 'Option 2'
    }, {
      value: 'option3',
      label: 'Option 3'
    }
  ]
})
```
