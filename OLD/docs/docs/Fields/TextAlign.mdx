# TextAlign

Generate a radio field to select multiple text alignments

## Options

```js
TextAlign('fieldName', {
  label: 'Label of the field',          // (optional)
  help: 'Some additional informations', // (optional)
  default: 'Default value',             // default to '' (optional)
})
```
