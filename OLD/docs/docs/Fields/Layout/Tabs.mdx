# Tabs

Group fields using tabs

## Usage

```ts
Tabs({
    label: "Content",
    fields: [
      Text("text1", { label: "Text 1" }),
      Text("text2", { label: "Text 2" }),
    ],
  },
  {
    label: "Settings",
    fields: [Text("text3", { label: "Text 3" })],
  }
)
```
