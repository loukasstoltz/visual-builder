# Checkbox

Generate a simple checkbox to toggle a value

## Options

```ts
Checkbox('fieldName', {
  label: 'Label of the field',          // (optional)
  help: 'Some additional informations', // (optional)
  default: false,                       // default to false (optional)
})
```
