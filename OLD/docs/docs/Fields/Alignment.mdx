# Alignement

Generate a radio field to select multiple alignments

## Options

```js
Alignment('fieldName', {
  label: 'Label of the field',          // (optional)
  help: 'Some additional informations', // (optional)
  default: 'Default value',             // default to '' (optional)
  vertical: false                       // Allow vertical align, default false (optional)
})
```
