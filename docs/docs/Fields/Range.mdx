# Range

Generate a slider to choose a numeric value

## Options

```js
Range('fieldName', {
  label: 'Label of the field',          // (optional)
  help: 'Some additional informations', // (optional)
  default: 0,                           // default value to 0 (optional)
  min: 0,                               // default to 0 (optional)
  max: 100,                             // default to 100 (optional)
  step: 1                               // default to 1 (optional)
})
```
