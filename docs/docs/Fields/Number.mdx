# Number

Generate a simple number field

## Options

```ts
Number('fieldName', {
  label: 'Label of the field',          // (optional)
  multiline: true,                      // Allow multiple line of text (default to false)
  help: 'Some additional informations', // (optional)
  default: 'Default value',             // default to ''
})
```
